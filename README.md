## Getting Started üèÅ


### Clone this repository

```bash
git clone https://github.com/DeVuDeveloper/salary-calculation.git
```

### Move into the directory with

```bash
cd salary-calculation
```

### Install linter

```bash
yarn install
```

### Start server

```bash
yarn start
```

### Run linter

For React JS:

```bash
 npx eslint .
```
